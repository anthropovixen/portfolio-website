WEB DEV MEETUP

WEB DEV MEETUP is a serverless, progressive web application (PWA) built with React using Google Calendar API to fetch upcoming events from a calendar built by Career Foundry. Users can search for a city and view a list of events hosted in the selected city. The app also allows users to visualize charts with the number of events per city and with percentage distribution of events per city by subject.

MOVIES MYFLIX APP

MOVIES MYFLIX APP is a RESTful API web application that provide users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favourite movies.
